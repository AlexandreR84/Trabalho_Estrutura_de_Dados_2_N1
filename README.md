# Analise de Desempenho de Algoritmos de Ordenacao

Projeto em Python para comparar Bubble Sort, Selection Sort e Insertion Sort em duas estruturas:

- Arrays
- Listas ligadas implementadas manualmente

## Como executar

```bash
python main.py
```

## Parametros opcionais

```bash
python main.py --benchmark-size 10000 --plot-sizes 5000 10000 50000
```

Os arquivos gerados sao salvos na pasta `saida/`.
